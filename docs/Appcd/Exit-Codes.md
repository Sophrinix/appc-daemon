## Exit Codes

| Code  | Description                                          |
| :---: | :--------------------------------------------------- |
| 0     | Success                                              |
| 1     | An error occurred                                    |
| 2     | Showed help screen                                   |
| 3     | Server was not running                               |
| 4     | Server is already running                            |
| 5     | Server was run as root                               |
| 6     | Configuration setting is read-only                   |
| 7     | Server must be restarted for changes to take effect  |
